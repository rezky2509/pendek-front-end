🔗 Pendek

Pendek means short in Malay. The word inspire me as it's always in my mind 😁

A sleek, minimalist URL shortener built with Next.js.

Pendek is a modern web application designed to turn long, clunky URLs into neat, shareable links. This project focuses on a fast, responsive, and intuitive user experience using the latest React features.
✨ Features

    Fast Performance: Optimized with Next.js for near-instant load times.

    Instant Shortening: Transform links in a single click using [Insert API Name].

    Copy to Clipboard: One-tap functionality to grab your new link.

    Responsive Design: Mobile-first approach for shortening links on the go.

🛠️ Built With

    Framework: Next.js 15 (App Router)

    Language: TypeScript 

    Styling: Tailwind CSS, ShadCN

    Deployment: AWS Amplify
